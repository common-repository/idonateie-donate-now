== Description ==

Donate Now is available to iDonate.ie members only. Please do not install if you are not a member of iDonate.ie.
Plugin creates a Donate button which links to your custom donation page. 

To update the colours of the page , go to your iDonate.ie admin account -> Widget Customisation. 

You can also enter a URL to redirect user following successful donation.

Visit www.idonate.ie for more details. 

For Support please email alan@idonate.ie


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `idonateie-donate-now` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Donate Now to create shortcodes for use on pages or code snippets for use in templates.
